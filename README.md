## To run the project: 👇

1. Clone the repository

```
git clone https://github.com/JhonEsteban/task-react-app.git
```

2. Install the node modules

```
yarn add
```

3. Create the .env file and add the required environment variables

```
REACT_APP_API_BASE=
```

4. Start the dev server

```
yarn start
```
